# Eaton Intelligent Power Manager Editions 3.0.0 - Release note
(June 2026)

## Scope:

* MONITOR/MANAGE/OPTIMIZE Editions

## Packaging:

IPM Editions 3.0.0 is delivered in the following formats:

* OVA package for VMware workstation 25 or VMware 7.0 (U3) or higher
* OVA package for VirtualBox 7.2.x
* OVA package for Hyper-V versions 2025, 2022, 2019
* OVA package for Proxmox versions 9.1.x, 8.4.x
* Upgrade package for existing customers
