# Eaton Intelligent Power Manager Editions 2.8.4 - Release note
(January 2026)

## Scope:

* MONITOR/MANAGE/OPTIMIZE Editions

## Packaging:

IPM Editions 2.8.4 is delivered in the following formats:

* OVA package for VMware workstation 17 or VMware 7.0 or higher
* OVA package for VirtualBox 7.1.x
* Hyper-V virtual appliance package (current supported version 2025, 2022, 2019)
* Upgrade package for existing customers

## Virtualization Connector Updates:
* Update VMware Connector to support latest version VMware 9.0
* Update VxRail Connector to support latest version VxRail 8.0

## Cybersecurity:
* IPM Editions now operates on the most recent stable release of the Debian OS (Debian 13)
* Latest package updates and security patches

## Fixes:
* Enable quick access to Virtualization from the main menu
* Resolved issues in Dynamic Group filtering
* ESXi credentials are now optional on vCenter connector creation

## Known issues:
* When the VMware connector (using SRM) is deleted, the SRM automation does not recognize the connector deletion, and no error message is displayed in automation UI
